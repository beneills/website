Software

This is hosted as a Jekyll blog, but makes use of other Jekyll features far more than the blogging stuff. The source is available on Github. If I wrote the site again, I'd investigate Octopress for apparently easier site admin (although it's pretty easy as-is). The resultant static HTML directory tree is easy to host anywhere. I've moved away from Github Pages, since I use custom plugins1.

I use JVectorMap to draw my places map, alongside JQuery for other stuff.

Hosting

Hosted with nginx on my very own 1GB Linode instance in London. I host other projects here also.
